## Overview
This repository contains a Vulnerability Assessment Report completed as part of the
Cyber Security Internship Program by Future Interns (2026).

The objective of this task is to analyze a public-facing website for common security
misconfigurations and vulnerabilities using ethical, read-only techniques, and to
present the findings in a clear, business-friendly manner.

## Target Website
- Name: OWASP Juice Shop (Demo Application)
- Type: Public demo web application
- Scope: Public-facing pages only

## Scope & Ethics
The assessment was conducted under strict ethical guidelines:
- Passive analysis only
- No exploitation or active attacks
- No authentication bypass
- No brute force or denial-of-service attempts

This assessment simulates a security audit, not a penetration test.

## Tools Used
- Nmap (basic port and exposure analysis)
- OWASP ZAP (passive scan)
- Browser Developer Tools (headers, cookies, client-side inspection)

## Deliverables
- Professional Vulnerability Assessment Report (PDF)
- Supporting evidence and screenshots
- Risk classification and remediation recommendations

## Disclaimer
This assessment was conducted for educational purposes only on a permitted demo
application. No real systems or data were harmed during this assessment.
